**[Play the WebGL Demo](https://abxdnego.github.io/knife-3d-unity-webgl/)**

## Technical Stack

- **Engine:** Unity with Universal Render Pipeline (URP)
- **Target Platform:** WebGL (Desktop & Mobile responsive)
- **Rendering:** PBR (Physically Based Rendering)
- **Textures:** High-fidelity PBR maps
- **Build Version:** 0.1.0

## Asset Features

### PBR Texturing

- **Albedo (Diffuse):** Color and material identity
- **Normal Maps:** Detailed surface geometry
- **Metallic Maps:** Surface reflectivity properties
- **Roughness/Smoothness Maps:** Surface finish variation
- **Ambient Occlusion:** Cavity and contact shadows

## Pipeline Overview

This asset was created following a complete technical artist workflow:

1. **High-Poly Modeling** - Detailed blade and handle geometry
2. **Topology Optimization** - Clean, game-ready mesh topology
3. **UV Unwrapping** - Efficient texture space layout
4. **Texture Painting** - PBR material creation (Substance Painter / Marmoset)
5. **Baking** - Normal maps and ambient occlusion baking
6. **Unity Integration** - URP material setup with proper texture assignments
7. **Optimization** - Polygon reduction and performance tuning
8. **Web Deployment** - WebGL build and GitHub Pages hosting
